## Preparation

* `npm install` (one time only)

## Start Arketype

* `npm run start`
* Goto [http://localhost:4000](http://localhost:4000)

## Selenium tests on BrowserStack

* `npm install` (one time only)
* update `/selenium/src/user.json` with credentials 
* `npm run test`
